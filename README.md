# Kawaiimon

Mini jeu web en JavaScript : clique sur un maximum de Kawaiimon en 30 secondes.

## Lancer en local

```bash
python3 -m http.server 8000
```

Puis ouvre <http://localhost:8000>.

## Règles

- Clique sur les créatures avant qu'elles disparaissent.
- Chaque capture vaut 1 point.
- À la fin, un rang est attribué selon le score.
